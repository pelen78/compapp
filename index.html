INTRO TO HTML

What is HTML?

The language used to create and structure web pages. It provides the basic building blocks like headings, paragraphs, and images, acting as the skeleton of the web page. CSS (Cascading Style Sheets):The language used to style and design web pages.

STUDENT ACTIVITY

Build an Informational Web Page: “Intro to HTML”

GOAL

You will create a real web page in Visual Studio Code that explains what HTML is and how it works.

You will ONLY use:

  • headings
  • paragraphs

No images, no lists, no links.

STEP 1 — Create your project

1. Open Visual Studio Code

2. Create a folder named: html_intro_project

3. Create a file inside it: index.html

STEP 2 — Add the basic HTML structure

Copy this first:

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Intro to HTML</title>
  </head>

  <body>

  </body>
</html>

STEP 3 — Build your informational page

Inside <body>, create a page that teaches someone about HTML.

REQUIRED SECTIONS

1) Main Title

- One <h1>

Examples: Intro to HTML, What is HTML?, Learning HTML

2) What is HTML?

- <h2>

- 1 paragraph explaining: what HTML stands for and what it is used for.

3) Why is HTML important?

- <h2>

- 1 paragraph explaining: why websites need HTML and what happens without it.

4) Headings in HTML

- <h2>

- 1 paragraph explaining: why headings exist and how they organize information.

Then add: <h3>, <h4>, <h5> (with text explaining what each level could be used for).

5) Paragraphs in HTML

- <h2>

- 1 paragraph explaining: why paragraphs matter and what happens if everything is one big text block.

STEP 4 — Open it in a browser

- Save file

- Open index.html

- Make edits → refresh → observe changes
