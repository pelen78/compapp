INTRO TO HTML

What is HTML?

The language used to create and structure web pages. It provides the basic building blocks like headings, paragraphs, and images, acting as the skeleton of the web page. CSS (Cascading Style Sheets):The language used to style and design web pages.

STUDENT ACTIVITY

Build an Informational Web Page: “Intro to HTML”

GOAL

You will create a real web page in Visual Studio Code that explains what HTML is and how it works.

You will ONLY use:

  • headings
  • paragraphs

No images, no lists, no links.

STEP 1 — Create your project

1. Open Visual Studio Code

2. Create a folder named: html_intro_project

3. Create a file inside it: index.html

STEP 2 — Add the basic HTML structure

Copy this first:

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Intro to HTML - Student Activity</title>
</head>

<body>

    <h1></h1>

    <h2></h2>
    <p>


    </p>

</body>

</html>

STEP 3 — Build your informational page

Inside <body>, create a page that teaches someone about HTML.

REQUIRED SECTIONS

1) Main Title

- One <h1>

Examples: Intro to HTML, What is HTML?, Learning HTML

2) What is HTML?

- <h2>

- 1 paragraph explaining: what HTML stands for and what it is used for.

3) Why is HTML important?

- <h2>

- 1 paragraph explaining: why websites need HTML and what happens without it.

4) Headings in HTML

- <h2>

- 1 paragraph explaining: why headings exist and how they organize information.

Then add: <h3>, <h4>, <h5> (with text explaining what each level could be used for).

5) Paragraphs in HTML

- <h2>

- 1 paragraph explaining: why paragraphs matter and what happens if everything is one big text block.

STEP 4 — Open it in a browser

- Save file

- Open index.html

- Make edits → refresh → observe changes

ACTIVITY 2: Continue Activity 1: Add Basic CSS

Goal: Keep your same index.html. Create a new styles.css file and add simple text styling.

Step 1 – Create your CSS file

Inside your project folder, create a new file named: styles.css

Step 2 – Link your CSS

Open index.html. Inside the <head> section, add this line:

<link rel="stylesheet" href="styles.css">

Save your file and refresh your browser. Nothing will change yet — that is OK.

Step 3 – Add basic styling

Open styles.css and copy this code:

body {
    font-family: Arial, sans-serif;
    background: #f5f7fb;
    color: #111827;
    line-height: 1.6;
    padding: 24px;
}
h1 {
    font-size: 34px;
}
p {
    font-size: 16px;
}

Stop here. Make sure your page looks cleaner and easier to read.

Extra Challenge: Individual Heading Colors

If you want each h1, h2, or h3 to have a different color, you can use the :nth-of-type selector in your CSS. Copy this example into your styles.css to see it in action:

h1 {
    font-size: 34px;
    color: rgb(184, 0, 0); /* Base color for all h1s */
    text-align: start;
}

h1:first-of-type {
    color: rgb(255, 0, 0);
}

h1:nth-of-type(2) {
    color: blue;
}

h1:nth-of-type(3) {
    color: green;
}

Note: You can follow this same pattern for your h2 and h3 tags to give them unique colors as well!
